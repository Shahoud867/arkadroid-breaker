## HOW TO BUILD & RUN

### Building the Project

**Option 1: Using Makefile** (Recommended)
```bash
cd "OOP Finalproject"
make clean
make
./Arkadroid          # Windows: Arkadroid.exe
```

**Option 2: Using Build Batch** (Windows)
```batch
cd OOP_Finalproject
build.bat
Arkadroid.exe
```

**Option 3: Manual g++ Compilation**
```bash
cd "OOP Finalproject"
g++ -std=c++17 *.cpp -o Arkadroid -lsfml-graphics -lsfml-window -lsfml-system
./Arkadroid
```

### Running the Game
```
./Arkadroid
```

**Note**: Game window is 800x640 pixels. Menu system navigable with number keys (1-4) or mouse.

## GAME FEATURES

### Level Progression
1. **Level 1**: Rectangle brick pattern, single ball, 1.0x speed
2. **Level 2**: Staircase brick pattern, single ball, 1.3x speed
3. **Level 3**: Recursive fractal pattern, dual paddles, 1.6x speed

### Power-ups (All 5 Implemented)
1. **Green Food** (triangle) → Double paddle width
2. **Pink Food** (square) → Halve paddle width
3. **Blue Food** (circle) → Slow ball ×0.5 for 5s
4. **Red Food** (rectangle) → Speed ball ×1.5 for 5s
5. **Yellow Food** (square) → Spawn 2 extra balls for 5s

### Controls
- **Mouse**: Move lower paddle
- **W key**: Upper paddle move left (Level 3)
- **R key**: Upper paddle move right (Level 3)
- **P / Escape**: Pause/Resume
- **Menu Keys**: 1=New, 2=Resume, 3=History, 4=Quit
- **Mouse**: Click to navigate all screens

## KNOWN LIMITATIONS & ASSUMPTIONS

1. **Font Loading**: Requires Windows system fonts (Arial, Calibri, Segoe UI)
   - Fallback chain ensures game works even if one font missing
   
2. **Ball Speed**: Multipliers apply multiplicatively, not additively
   - Blue: ×0.5, Red: ×1.5 (stackable but not simultaneous)
   
3. **Paddle Size**: Constrained between 30px (min) and window-width-20px (max)
   - Prevents paddle becoming too small or too large
   
4. **Multi-ball Timeout**: Extra balls deactivate after 5 seconds
   - Original ball always remains active
   - Prevents infinite ball multiplication
   
5. **Level Transition**: Lives reset to 2 at each level
   - Score accumulates across all levels
   - No penalty for level progression

## SUPPORT & TROUBLESHOOTING

### Build Issues
**Error**: `lsfml-graphics: command not found`
- **Solution**: Ensure SFML development libraries are installed
- Windows: Download from sfml-dev.org, add to compiler search path

**Error**: Compilation timeout
- **Solution**: Run `make clean` first, then `make`

### Runtime Issues
**No fonts appear on screen**
- **Solution**: Check `C:\Windows\Fonts\` for arial.ttf or alternatives
- Game has fallback chain, but requires at least one font

**Ball gets stuck**
- **Solution**: This shouldn't happen. Report as bug if occurs.
- Delta time capped at 0.1s to prevent physics instability
